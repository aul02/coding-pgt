CHKDSK = untuk memeriksa kondisi harddisk
DXDIAX = untuk melihat informasi spesifikasi komputer
PING = untuk memeriksa koneksi internet ke suatu website atau alamat IP
EXIT = untuk keluar dari aplikasi CMD
HELP = untuk meminta bantuan informasi cara penggunan commed promt
DATE = untuk menampilkan tanggal pada layar cmd 
VER  = untuk menampilkan informasi sistem
TIME = untuk menampilkan waktu,tanggal pada layar CMD 
START = untuk membuka jendela CMD baru 
WHOAMI = untuk melihat nama dari user yang ada pada komputer 
SYSTEMINFO  = untuk menampilkan informasi komputer 
TASKLIST = untuk menampilkan informasi aplikasi yang sedang berjalan
COLOR  = untuk mengganti warna pada cmd
FTYP = untuk menampilkan tipe file berserta informasi
DRIVERQUERY = untuk melihat daftar apa saja yang terinstal 
NETSTAT = untuk membuka semua daftar port yang terbuka 
CLS = untuk mengmbalikan seperti semula 
DIR = untuk menampilkan file dan folder yang aktif
TREE = untuk menampilkan struktur dalam satu folder
CHDIR = untuk mengetahui folder yang sedang aktif
GETMAC = untuk melihat alamat mac atau alamat fisik dari komputer
